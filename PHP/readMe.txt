
Voici quelques exercices pratiques en PHP pour t'aider à mettre en pratique ce que tu viens d'apprendre. Je vais les organiser en trois niveaux pour t'aider à progresser.

    Exercices de Niveau Débutant

1. Hello World avec Variables
Crée un script PHP qui :
- Déclare une variable $prenom et assigne-lui ton prénom.
- Affiche le message : "Bonjour, {prénom} !" en utilisant echo.

2. Addition Simple
- Crée un fichier PHP qui :
- Déclare deux variables $a et $b avec des valeurs numériques.
- Affiche la somme de $a et $b.

3. Conditions d'Âge
Crée un fichier PHP qui :
- Demande à l'utilisateur d'entrer son âge (par exemple via un champ dans un formulaire).
- Affiche un message indiquant si la personne est mineure ou majeure.

4.Calculatrice de Surface de Rectangle
Crée un script qui calcule la surface d’un rectangle :
- Crée deux variables, $largeur et $longueur, et assigne-leur des valeurs.
- Affiche la surface en multipliant largeur par longueur.


    Exercices de Niveau Intermédiaire

5. Table de Multiplication
Écris un script PHP qui affiche la table de multiplication de 1 à 10 pour un nombre donné :
- Déclare une variable $nombre.
- Utilise une boucle for pour afficher le résultat de chaque multiplication de 1 à 10 avec ce nombre.

6. Nombre Pair ou Impair
Crée un fichier PHP qui :
- Demande un nombre à l'utilisateur (via un formulaire).
- Indique si le nombre est pair ou impair.

7. Calculateur de TVA
Crée un script qui calcule le montant total avec TVA pour un prix donné :
- Demande à l'utilisateur de saisir un montant.
- Applique un taux de TVA (par exemple, 20 %) et affiche le montant total avec TVA.

8. Convertisseur de Devises
Crée un script PHP qui convertit des euros en dollars :
- Demande à l'utilisateur de saisir un montant en euros.
- Applique un taux de conversion fixe (par exemple, 1 € = 1,10 $).
- Affiche le montant converti en dollars.


    Exercices de Niveau Avancé

9. Gestion de Liste de Courses
Crée un fichier PHP qui :
- Contient un tableau avec des éléments de liste de courses (ex : "Pommes", "Bananes", "Lait").
- Affiche chaque élément de la liste de courses dans une balise HTML <li>.
- Permet d'ajouter un nouvel élément via un formulaire.

10. Calcul de Moyenne de Notes
Crée un script PHP qui :
- Permet à l'utilisateur d'entrer cinq notes (via un formulaire).
- Calcule la moyenne des notes et affiche le résultat avec un message indiquant si l'utilisateur a "validé" ou "non validé" (par exemple, si la moyenne est supérieure à 10/20).

11. Mini-Annuaire
Crée un script PHP qui :
- Demande le nom, prénom et numéro de téléphone d'une personne via un formulaire.
- Ajoute cette personne à un tableau associatif.
- Affiche la liste complète des personnes enregistrées, avec nom, prénom et numéro de téléphone.

12. Convertisseur de Température
Écris un script PHP qui :
- Permet à l'utilisateur de saisir une température en Celsius via un formulaire.
- Convertit cette température en Fahrenheit avec la formule : F = (C * 9/5) + 32.
- Affiche le résultat de la conversion.
